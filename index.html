Overview

The “Programming with Big Data in R” project (pbdR) enables high-level distributed data parallelism in R, so that it can easily utilize large HPC platforms and begin to use multiple levels of parallelism while retaining its original convenience for mathematics of data. We achieve this through, in part, providing a simplified interface to MPI as well as a tight coupling with distributed dense linear algebra software (PBLAS and ScaLAPACK, and their MPI library BLACS). The routines in these libraries are engaged through R’s classes and methods, so that the R language syntax is unchanged. While the user is not asked to deal with the details of managing the data distribution and MPI communication, the user is asked to be aware of the data distribution and provided with high-level functions to manage it if needed.

To this end, we provide several R packages which focus on analyzing big data, especially on large distributed machines. The packages are meant to be used in the SPMD programming model, with special focus on large scale computing clusters. Packages and instructions for installation are available in the Download tab.
