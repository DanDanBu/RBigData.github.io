General

pbdMPI

pbdMPI provides an efficient interface to MPI by utilizing R’s S4 classes and methods with a focus on Single Program, Multiple Data (SPMD) parallel programming style, which is intended for batch parallel execution. pbdMPI functions are SPMD extensions of MPI functions, rewritten from Rmpi with many performance improvements, as well as a simplified interface.

The pbdMPI package:

  • Provides high-level, abstract interface to low-level, fast MPI code.
  • Avoids potential dead locks in an interactive model.
  • Gains performance from directly calling MPI functions.
  • Reduces the cost of compression in communication.
  • Offers easier programming and debugging efforts.

Computation

pbdSLAP

pbdSLAP provides access to low-level, distributed matrix algebra Fortran libraries for use with R. Namely, the package offers the R user access to a subset of ScaLAPACK, PBLAS, and BLACS routines, which are utilized in the pbdBASE and pbdDMAT packages.

The pbdSLAP package:

  • Simplifies installation for the higher level pbdR packages.
  • Allows the R user access to fast Fortran libraries for distributed linear algebra.
  • Provides a simple infrastructure for using other distributions of ScaLAPACK, such as Intel’s MKL, within R and the pbdR toolchain (see package vignette for details).
pbdBASE

pbdBASE offers a set of S4 classes and methods for dealing with distributed objects. At present, this includes a distributed dense matrix class, which manages block-cyclically distributed data for use with ScaLAPACK. Most of the low-level details have been abstracted away for the typical user, while access to the full complexity of BLACS is available to advanced users familiar with ScaLAPACK.

The pbdBASE package:

  • Offers a set of S4 classes and methods for storing and manipulating distributed data.
  • Wraps essential low-level functions to utilize pbdSLAP in the pbdR toolchain.
  • Utilizes a specialized MPI communicator for use with ScaLAPACK.
pbdDMAT

pbdDMAT is a set of computational methods for the distributed matrix class object defined in pbdBASE which attempt to mimic native R syntax as much as possible. Currently, these methods include everything from abs() and apply() to svd() and cov().

The pbdDMAT package:

  • Offers methods which make computations in R scalable.
  • Gives ScaLAPACK speed with R syntax.
  • Abstracts away difficult MPI programming techniques, while still offering great performance and scaling.
  • Breaks R’s local memory cap for matrix objects by distributing the data across many processors.
